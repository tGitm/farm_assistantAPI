FarmLog API

Dobrodošli v API-ju aplikacije FarmLog!

Aplikacija je namenjena lažjemu arhiviranju in pregledovanju opravljenega dela na kmetijskih zemljiščih zemljiščih.

Razvil: Tim Rus ©2022

User Routes

  • Registracija novega uporabnika -> POST https://farmlog-api.vercel.app/api/user/register

  • Prijava uporabnika -> POST https://farmlog-api.vercel.app/api/user/login

  • Pridobi uporabnika -> GET https://farmlog-api.vercel.app/api/user/:id

  • Posodobi uporabnika -> PUT https://farmlog-api.vercel.app/api/user/edit/:id

Land Routes

  • Pridobi vsa zemljišča -> GET https://farmlog-api.vercel.app/api/get-lands

  • Pridobi vsa zemljišča po MID -> GET https://farmlog-api.vercel.app/api/get-geometry/:id

Chore Routes

  • Pridobi vsa opravila uporabnika -> GET https://farmlog-api.vercel.app/api/chore/get-chores/:id

  • Pridobi opravilo preko id-ja -> GET https://farmlog-api.vercel.app/api/chore/get/:id

  • Posodobi določeno opravilo -> PUT https://farmlog-api.vercel.app/api/chore/update-land-work/:id

  • Odstrani zemljišče -> DELETE https://farmlog-api.vercel.app/api/chore/delete/:id
